# opis komponentu
Komponent do obsługi ksef API dla laravel

# zadania
    - przygotowanie class do logowania, pobierania faktury, wystawiania faktur, faktury w bazie będą przechoweywane za pomocą 
    - klasa do odczytania zaszyfrowanych faktur


# faktury 
faktury w bazie zostaną zaszyfrowane w formie xml za pomocą klucza z .env tak jak dane do logowania, nie szyfrowany będzie numer faktury, data kupna sprzedaży, czy to jest kupno czy sprzedaż, nip i nazwa kupującego i sprzedającego

# dane logowania
Dane logowania i certyfikaty, tokeny, hasło do certyfikatu zostaną zapisane w bazie danych w postaci zaszyfrowanej, dane deszyforwane będą za pomocą klucza
